# UNP_PROJECT_E-COMMERCE_DATA_ANALYSIS_USING_MYSQL

I did this project in collaboration with UNP, where the primary objective was to develop a robust e-commerce database system using MySQL for efficient data management and analysis of business operations. The system was designed to handle core functionalities and support data-driven decision-making.
I designed a well-structured relational database by identifying key entities and defining relationships between them. Primary and foreign key constraints were implemented to ensure data integrity, consistency, and efficient querying across multiple interconnected tables.
The database included tables such as Products, Customers, Orders, Payments, Returns, and Shipments. I developed and managed these tables while maintaining proper relationships, enabling smooth handling of various business processes within the system.
I implemented data insertion and simulated real-world scenarios including order processing, payment tracking, shipment updates, and product returns. I also performed advanced data analysis using SQL queries involving joins, aggregations, grouping, and filtering to extract meaningful insights like sales trends, customer behavior, and revenue patterns.
This project helped me gain practical experience in designing databases, writing efficient SQL queries, and deriving meaningful insights to solve real-world business problems.
